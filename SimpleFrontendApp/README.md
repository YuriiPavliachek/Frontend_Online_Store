# Blazor Online Shop 🛒

Це простий інтернет-магазин на **Blazor**, де можна переглядати товари, додавати їх у кошик та робити вхід/реєстрацію користувача.

---

## 🚀 Особливості

- Головна сторінка з товарами  
- Кошик збоку (sidebar), який автоматично оновлюється при додаванні товарів  
- Підрахунок загальної вартості  
- Сторінка **Кошик** з кнопкою “Оплатити”  
- Система авторизації та реєстрації користувачів  
- Сторінка **Профіль** доступна тільки після входу  
- Динамічна панель навігації: кнопка “Вхід” зникає після логіну

---

## 🛠 Технології

- [Blazor WebAssembly](https://dotnet.microsoft.com/apps/aspnet/web-apps/blazor)  
- C# / .NET 8  
- HTML / CSS / Bootstrap
